## Project Overview

**ArabicInSanskrit** is a research repository for studying Arabic loanwords occurring in the Cologne Digital Sanskrit Lexicon dictionaries. Work is organized per-dictionary, primarily through the [GitHub issue tracker](https://github.com/sanskrit-lexicon/ArabicInSanskrit/issues).

Each dictionary's Arabic loanword analysis is tracked as a separate issue, identifying entries where Arabic-origin terms appear in Sanskrit context.

## Common Commands

No standalone scripts — analysis work is conducted through issues and manually applied corrections to `csl-orig` source files via the standard `updateByLine.py` pattern.
